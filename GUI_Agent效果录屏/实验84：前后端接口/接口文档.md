## POST 创建XRWF

POST /violation

> Body 请求参数

```json
{
  "name": "string",
  "start_time": "string",
  "end_time": "string",
  "address": "string"
}
```

### 请求参数

|名称|位置|类型|必选|中文名|说明|
|---|---|---|---|---|---|
|body|body|object| 否 ||none|
|» name|body|string| 是 | 应用名称|none|
|» start_time|body|string(yyyy-MM-dd HH:mm:ss)| 是 | 开始日期|none|
|» end_time|body|string(yyyy-MM-dd HH:mm:ss)| 是 | 结束日期|none|
|» address|body|string| 是 | 地址|none|

> 返回示例

> 200 Response

```json
{
  "code": "0",
  "msg": "success"
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|string|true|none||none|
|» msg|string|true|none||none|
