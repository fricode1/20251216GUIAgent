## Playwright (Codegen)

安装后运行 npx playwright codegen [网址]。

它会打开一个浏览器窗口，你在网页上的所有点击、输入都会自动生成 Python/JavaScript/Java 代码。

```
npx playwright codegen --target python
```

回放： 直接运行生成的脚本即可完美复现。

## 如何在内网运行 

```
python -m playwright codegen
```
