# 实验背景

我开发了web应用。前端就是一个最简洁的chatbot。能够接受自然语言输入。

后端是基于 自然语言输入 - AI意图理解 - AI任务分发 - 浏览器自动化操作视综平台/OCR识别图像中文本 链条的行为交通违章查询。

这激发了我的想象力。该平台既然能做1项任务，说明也能做 100 项任务。

公安业务系统平台非常多，有视综平台、警综平台、情指行平台、接处警平台，等等等等。

这些平台操作很复杂，基层民警使用不便。

还有些平台的重复性工作，比如，有数千张行人违法抓拍照，需要逐一到视综平台确认每个抓拍人员的身份，需要耗费基层民警的大量时间精力。

# 实验目的

评估建设 统一平台 的可行性
