## 快速复制代码

### 【成功】设置docker不用sudo权限，从而用vsode操作容器文件

sudo groupadd docker
sudo usermod -aG docker $USER
newgrp docker
sudo chown "$USER":"$USER" /home/"$USER"/.docker -R
sudo chmod g+rwx "$HOME/.docker" -R
docker run hello-world

重启电脑

vsode启动容器时，确保代理能访问外网
