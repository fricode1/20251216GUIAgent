# 实验背景

对于真正的自动化任务，web agent 就是个鸡肋。真正干活还得是RPA。如果这事做成了，南阳要用，难道把八爪鱼发给他们？不得整个开源的包装成自己的？所以需要找开源RPA。

# 实验目的

找最好的的开源RPA。希望像八爪鱼那样能有交互界面。

# 实验步骤
