# 实验目的

# 实验步骤

## 拷贝镜像

```powershell
docker save -o copaw.tar agentscope/copaw:latest
```

## 内网部署镜像

```bash
docker load -i copaw.tar.gz
```

## 运行容器

```bash
docker run -p 8088:8088 -v copaw-data:/app/working agentscope/copaw:latest
```

注意：8088:8088的含义是，第一个为宿主机的端口，第二个为容器内的端口。如果宿主机的端口已经被占用，则要修改宿主机端口，不能修改容器端口。
