## 看看哪种 pip install 不需要 pytorch

- cnocr
    - 需要pytorch
- pip install easyocr
    - 需要pytorch
- pip install paddlepaddle  # CPU版本
    - 不需要pytorch
    - pip install paddleocr
        - 不需要pytorch
    - NotImplementedError: (Unimplemented) ConvertPirAttribute2RuntimeAttribute not support [pir::ArrayAttribute<pir::DoubleAttribute>]  (at ..\paddle\fluid\framework\new_executor\instruction\onednn\onednn_instruction.cc:118)
    - 导出为ONNX：https://github.com/PaddlePaddle/PaddleOCR/blob/main/deploy/paddle2onnx/readme_ch.md
- OnnxOCR
    - 低版本numpy安装失败
    - 高版本numpy运行失败
- RapidOCR
    - pip install rapidocr onnxruntime
