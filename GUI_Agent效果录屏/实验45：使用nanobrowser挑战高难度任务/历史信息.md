        Executor->>Planner: execute()
        
        activate Planner
        Planner->>MsgMgr: getMessages()
        Note over MsgMgr: 包含 System Instructions,<br/>User Request 和网页快照
        MsgMgr-->>Planner: messages (History + Current State)
        
        Planner->>LLM: invoke(plannerMessages)
        Note right of LLM: 这里的 AI 观察到：<br/>[33] <button>Starred 11.8k
        LLM-->>Planner: modelOutput (JSON)
        
        Planner->>Planner: filterExternalContent()
        Note over Planner: 提取 reasoning, observation,<br/>next_steps, action 等
        
        Planner-->>Executor: latestPlanOutput
        deactivate Planner

        Note over Executor: 检查 Plan 状态：<br/>发现任务已完成 (done: true)
        
        alt 任务完成
            Executor-->>Index: 返回结果 (Success)
        else 继续下一步
            Executor->>Executor: 执行 action (如 click, input_text)
        end
    end
